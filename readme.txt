Данная программа создана исключительно в целях обучения и не предназначена для действий, связанных с нарушением законодательства, и причинением ущерба.

Для её работы необходим Tor и собственный телеграм бот, созданный через @BotFather .

Чтобы начать пользоваться, вводим нужные данные в файле config.py и bomber.py, затем запускаем файл bot.py и пишем в лс нашему боту команду /start
